# kuCity 城市选择jquery插件  
  

插件特性
----------  
1.单例  
2.支持搜索(借用了阿里的API)  
3.搜索框函数节流    
  
  
插件使用
----------  
一，引进插件（依赖jquery）  

`<link rel="stylesheet" href="kuCity.css"> `
  
`<script src="http://cdn.bootcss.com/jquery/1.9.1/jquery.min.js"></script>`
`<script src="kuCity.js"></script>`

二，使用  

`$('.search').kuCity();`
